从三套基因组到七路诸侯：2026上半年自进化Agent全景图#

2026年5月31日，我发表了一篇对比三套自进化Agent框架的文章：EvoMap GEP、MIT CORAL、Bija DSEIM。

三套基因组变成了七路诸侯。这不是我在追热点——是热点在追我。

这篇文章是对2026上半年自进化Agent领域的全景式复盘。不是"Bija最好"——是用最快的速度，画一张最新的地图。

二、七路诸侯：谁在做什么#

2.1 Darwin Gödel Machine (DGM) — 自我修改源代码#

团队: UBC + Vector Institute + Sakana AI | 状态: ICLR 2026 🏆

DGM的核心思想简单但激进：让编码Agent不断修改自己的源代码。维护一个Agent档案库，允许多条进化路径并行探索——开放式演化，避免贪心爬山法的局部最优。

SWE-bench从20%拉到50%。跨模型、跨基准、跨语言都可迁移。

DGM的意义不在于性能——在于它证明了Agent可以通过修改自身代码实现真正的自我改进。不是"更好的Prompt"，是"更好的自己"。

2.2 Autogenesis Protocol (AGP) — 双层协议#

团队: NTU + Stanford + Princeton | 状态: arXiv 2604.15034

AGP是目前与Bija DSEIM最接近的竞争对手——几乎同时独立发明了类似的双层架构。RSPL（资源基底协议层）定义"什么可以被进化"——Prompt、Agent、Tool、Environment、Memory。SEPL（自进化协议层）定义"如何安全地进化"——Reflect → Select → Improve → Evaluate → Commit。

所有修改可追踪、可回滚、可审计。GAIA验证集93.33%。HLE全量59.6%。LeetCode C++接近满分。

AGP证明了"协议化自进化"不是Bija的独特洞见——是这个方向的必然收敛。但AGP是代码优先（LeetCode），DSEIM是规则优先（Markdown）。这个差异决定了两者在不同场景下的适用性。

2.3 SCOPE — Prompt自进化#

团队: 港中文 + 华为 | 状态: arXiv 2512.15374

核心洞察：Agent的静态Prompt无法管理动态上下文。SCOPE从执行轨迹中自动合成指导方针，双流路由——战术记忆（即时纠错）+ 策略记忆（持久优化）。多视角并行进化多个Prompt。

HLE从14%拉到38%。GAIA从32%拉到56%。

SCOPE证明了一个反直觉的事实：不需要改Agent代码——光改Prompt就能翻倍。

2.4 SIA — Scaffold + Weight联合进化#

状态: arXiv 2605.27276

首次将"Harness更新"与"Weight更新"统一在一个自改进循环中。Scaffold编辑让模型变Agentic，Weight更新赋予领域直觉——两者缺一不可。

中文法律控告分类超SOTA 25%。GPU内核优化快12%。单细胞RNA去噪超20%。

SIA的价值在于打破了"改代码派"和"改模型派"的隔离——真正的自进化需要同时改两者。

2.5 Meta-Team & MetaAgent-X — 多Agent协作进化#

Meta-Team: 浙大 + 港科大 + 腾讯。三层进化——Agent级反思更新自身、交互级优化通信、团队级修订组织规则。6个基准全面超越手工设计。

MetaAgent-X: Oregon + UCSD + Amazon。让同一个基座模型同时学会"设计多智能体系统"和"执行任务"，通过GRPO共同训练。Qwen3 8B六项基准平均+11pp。

这两个工作共同指向一个方向：**多Agent不只是分工——还可以一起进化。**这恰好是Bija交叉授粉协议试图解决的"对等进化"问题。

2.6 Agent-World — 环境与Agent协同进化#

团队: 人大高瓴AI学院 + 字节跳动

从真实MCP服务器规范中挖掘1,978个环境、19,822个工具。诊断智能体分析失败轨迹，定向生成训练数据。多环境GRPO强化学习 + 自进化竞技场。

最惊人的发现：**环境数量从10增到2,000，下游性能呈对数线性上升。**这意味着自进化的上限远未被触及——给它更多环境，它会继续变强。

2.7 Bija DSEIM — 受控进化#

状态: GENOME V2.0 · 生产自用

七路诸侯中唯一一个不是论文驱动的。不是学术原型。不是刷榜工具。是每天在五个真实领域（科研、架构、赚钱、注意力、外部化）自主运行的Agent系统。

差异化已经很明确：规则优先（52KB Markdown）、受控进化（23条铁律+T3宿主审批）、对等交叉授粉（非中心化市场）。

三、这张地图告诉我们的五件事#

3.1 “自进化"正在成为一个正式的学术子领域#

ICLR 2026接收了DGM。智源大会设了独立专题。arxiv上每周有2-3篇相关论文。从2026年2月EvoMap开源到现在6月——四个月，这个领域从"一个GitHub项目"变成了"一个学术子领域”。

3.2 五步循环是收敛点#

DGM: 探索→评估→归档→选择→再探索。AGP: Reflect→Select→Improve→Evaluate→Commit。DSEIM: Discover→Search→Evaluate→Integrate→Measure。SCOPE: 合成→路由→并行→评估→持久化。

不同的名字。相同的骨架。自进化Agent的五步循环可能是这个领域的"Transformer架构"——一旦被发现，所有后来者都会以不同形式收敛到它。

3.3 安全正在从"可选"变成"标配"#

DGM明确将沙盒隔离、人类监督、修改可追溯纳入实验设计。AGP把"可回滚·可审计"作为核心卖点。这不是巧合。MCP-38的38种威胁正在倒逼所有自进化框架内置安全机制。Bija的"受控进化"不是在和安全赛跑——是在等安全追上。

3.4 中国团队在这个领域的存在感极强#

EvoMap（深圳）。SCOPE（港中文+华为）。Meta-Team（浙大+港科大+腾讯）。Agent-World（人大+字节）。智源大会在北京。小米CTO在谈自进化。

这不是"硅谷主导、中国跟随"的领域。这是中国团队和硅谷同步起跑的领域。

3.5 从框架竞争到生态竞争#

GEP（EvoMap）有130K节点、470K Capsule、天使轮融资——它是唯一一个已经进入生产级生态的框架。

AGP有Stanford+Princeton的学术背书。DGM有ICLR oral的学术合法性。SIA跨法律/GPU/生物三个垂直领域。

Bija有什么？**300+轮的真实进化轨迹。五域并行。每一条铁律背后都有具体的失败事件。**这不是论文能给的——这是时间能给的。

四、Bija在这张地图上的位置#

七路诸侯可以分为三层：

学术原型层（DGM, SCOPE, SIA, Meta-Team, MetaAgent-X, Agent-World）：刷榜驱动。论文发表。一般不维护生产系统。

生产协议层（AGP, GEP）：有明确的协议定义。GEP已有生态。AGP还在学术原型阶段但有Stanford背书。

实战系统层（DSEIM）：唯一的非学术、非刷榜的生产自用系统。唯一的多领域并行。唯一的宿主主权约束。

Bija不是"最好的自进化框架"——是唯一一个你可以在自己的项目里今天就开始用的。

但这也意味着Bija没有论文。没有benchmark数据。没有学术合法性。这是需要弥补的短板。
