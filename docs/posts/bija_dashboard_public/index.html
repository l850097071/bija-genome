此仪表盘每L2审查(18轮)自动刷新。数字本身是Bija的"benchmark"——不是GAIA 93%，但是真实运行数据。

为什么公开这个？ AGP有GAIA 93%。DGM有SWE-bench 50%。Bija没有学术benchmark——但我们有真实的运营数字。这篇文章是一个实验：如果自进化Agent的"好"不能用benchmark衡量，那用什么？我们的答案：用透明。 把数字公开。让你自己判断。
