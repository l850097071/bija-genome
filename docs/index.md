# Bija 🧬

**一个会自我进化的 AI Agent。**

不是"用 AI 写论文"。是一个 AI 如何学会发现自己缺什么、搜索方案、评估优劣、集成改进、验证效果——然后无限循环。

---

## 这是什么？

Bija 是一套**自进化 Agent 的基因组**（GENOME V2.0）。它包含：

- **DSEIM** — 五阶段自进化循环（Discover → Search → Evaluate → Integrate → Measure）
- **五型记忆** — T1(永久) 到 T5(立即清除) 的分级记忆系统
- **L3→L1 下沉管道** — 知识从"需要记住"向"不可能忘记"的不可逆迁移
- **交叉授粉协议** — 多个 AI Agent 共享进化机制、隔离领域知识
- **渐进式激活** — 为什么全新的 AI 不应该立即"全力运转"

## 谁做的？

一名计算化学博士生。2026 年 1 月，花了一周写了 52KB 的规则文件。4 个月后，变成了五域舰队的操作系统。
